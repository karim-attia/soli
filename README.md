# Free and open source Solitaire app.

No ads, in-app purchases, or gamification to get you to play more.

# Download

- [Download on the App Store for iPhone](https://apps.apple.com/app/id6755950876)
- [Get it on Google Play for Android](https://play.google.com/store/apps/details?id=ch.karimattia.soli)

# Features

- Klondike with draw 1 and undo.
- Auto-complete when all cards are face up.
- Solvable games
- Nice animations
- Undo time travel through game
